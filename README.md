# Nithan Aravinth - Interactive Digital Portfolio

> **Live Demo:** https://nithanaravinth.ca/#home

Welcome to the source code for my interactive digital portfolio. I am a Digital Media Game Arts student. 

This repository contains the front-end code for my personal website, engineered to showcase my work across complex 3D modeling, software development, and technical art.

## ✨ Key Technical Features

This portfolio was built entirely without front-end frameworks to demonstrate a strong foundational understanding of web technologies, DOM manipulation, and responsive design principles.

* **Integrated 3D Viewer:** Utilizes Google's `<model-viewer>` component bundled with a custom-written Vanilla JavaScript modal system to render `.glb` 3D assets directly in the browser.
* **Generative Canvas Background:** A custom, interactive particle network built using object-oriented Vanilla JavaScript on an HTML5 `<canvas>`. Features real-time mouse proximity detection and dynamic line-drawing algorithms.
* **CSS Architecture:** Uses CSS variables, Grid/Flexbox layouts, glassmorphism UI elements, and strict media queries for seamless cross-device responsiveness.
* **Backend Integration:** Serverless contact form handling via the Formspree API.
* **Hosting:** Deployed via GitHub Pages and secured through Cloudflare's CDN.

## 📁 Repository Structure

* `index.html` - The core structural file, utilizing section IDs for smooth scrolling navigation and embedded modal frameworks.
* `Portfolio_Website(CSS).css` - All styling, including the uniform project card grid and frosted-glass modal overlays.
* `Portfolio_Website(JS).js` - Contains the logic for the interactive canvas background and the dynamic 3D model viewer pop-ups.
* `models/` - Directory housing the optimized `.glb` 3D assets for web rendering.
* `Nithan_Aravinth_Resume_Website.pdf` - Embedded resume optimized for web viewing.
